# 6ボールパズル 仕様書

## 概要
「世界のアソビ大全51」収録の「6ボールパズル」をWebブラウザで遊べるように再現する。

## ゲームルール

### 基本ルール
- 画面上から**3つのボール**が三角形の形で落ちてくる
- ボールの色は**赤・青・紫・黄・紫の5色**（ランダム）
- **同色6つ**を繋げると消える
- 画面上部のラインを超えて積み上がったら**ゲームオーバー**

### わざ（特殊消し方）
| わざ名 | 形状 | 効果（対戦時） |
|--------|------|----------------|
| ストレート | 横または斜めに6つ直線 | おじゃまボール2列 |
| ピラミッド | 三角形（1-2-3 or 3-2-1） | おじゃまボール24個 |
| ヘキサゴン | 六角形（輪っか） | おじゃまボール36個 |

### フィールド
- 横: 10マス
- 縦: 12マス（＋上部に出現エリア3マス）

## 操作方法

| 操作 | キー |
|------|------|
| 左移動 | ← / A |
| 右移動 | → / D |
| ソフトドロップ（少し下げる） | ↓ / S |
| ハードドロップ（直下に落とす） | ↑ / W |
| 回転（時計回り） | Space |
| 回転（反時計回り） | Shift |
| ポーズ | P / Escape |

## 技術仕様

### 使用技術
- **HTML5 Canvas** - ゲーム描画
- **JavaScript (Vanilla)** - ゲームロジック
- **CSS** - UI/レイアウト

※ シンプルな2Dゲームなので、Unityは不要。ブラウザのみで完結。

### ファイル構成
```
six-ball-puzzle/
├── index.html      # メインHTML
├── style.css       # スタイル
├── js/
│   ├── main.js     # エントリーポイント
│   ├── game.js     # ゲームロジック
│   ├── board.js    # ボード管理
│   ├── piece.js    # 落下ピース管理
│   ├── input.js    # キー入力処理
│   └── renderer.js # 描画処理
└── SPEC.md         # この仕様書
```

### ゲームフロー
1. タイトル画面
2. ゲーム開始
3. ピース生成 → 落下 → 設置 → 消去判定 → 連鎖 → 次のピース
4. ゲームオーバー → スコア表示 → リトライ

### 落下速度
- 初期: 1秒に1マス
- レベルアップごとに速度上昇

### スコア計算
- 基本消去: 100点 × 消したボール数
- わざボーナス:
  - ストレート: ×1.5
  - ピラミッド: ×3
  - ヘキサゴン: ×5
- 連鎖ボーナス: 連鎖数 × 1.5倍
