# ykshio tools

便利なウェブツール集です。

## URL

https://ykshio.github.io/tool/

## ツール一覧

| ツール | 説明 |
|--------|------|
| [句読点変換](./punctuation-converter) | 文章の句読点を「、。」と「，．」で相互変換 |
| [改行削除](./deleteLF) | 文章中の不要な改行を削除・整形 |
| [ブロック崩し](./brickBreaker) | シンプルなブロック崩しゲーム |
| [QRコード一括生成](./qrcode) | 複数のURLからQRコードをまとめて作成 |
| [カラーテスト](./colortest) | カラーコードで遊べる |
| [A〜Hルーレット](./roulette) | A〜Hからランダムに1つ選ぶ |
| [全角チェッカー](./mitsuiesan) | 全角と半角の文字をチェック |
